# Line Charts Overview

Line charts display data as a series of points connected by straight or curved lines,
making them ideal for visualizing trends, patterns, and changes over continuous intervals such as time.
Charty provides several line chart variants for different data visualization needs.

## When to use line charts

- You need to visualize trends and changes over time or continuous intervals.
- You want to show how values evolve and identify patterns in sequential data.
- You need to compare multiple data series on the same timeline.
- You want to emphasize cumulative totals or composition over time.
- You need to show relationships between continuous variables.

## Choosing a line chart type

- Use a **Line Chart** for showing a single metric's trend over time.
- Use a **Multiline Chart** to compare multiple series on the same timeline.
- Use an **Area Chart** to emphasize magnitude and the "volume" of data over time.
- Use a **Stacked Area Chart** to show how multiple components contribute to a total over time.
- Use a **Point Chart** to analyze correlations and relationships between two variables.
- Use a **Bubble Chart** when you need to visualize three dimensions simultaneously (X, Y, and size).
- Use a **Combo Bar Chart** when you want both bars and lines in the same view.
